# Contributing to frontend

## Prerequisites
Node 20+ / Docker Desktop

## Getting started
git clone -> cp .env.example .env.local -> npm install -> npm run dev

## Branch naming
feat/... fix/... refactor/... chore/... docs/...

## Commit messages
Conventional Commits : imperative, lowercase, under 72 characters

## Workflow
branch -> code -> npm run build -> PR -> review -> merge

## Running the dev server
npm run dev

## Code style
TypeScript strict mode, Prettier defaults, Tailwind utility classes
